Hypothesis Testing

H0: The class of a passenger's ticket DOES NOT have an effect on their likelihood of survival.

HA: The class of a passenger's ticket DOES have an effect on their likelihood of survival.

Assume alpha value of 0.05, we'll apply a chi-square test because we want to see if there is a statistically significant relationship between the ticket class and the likelihood of the person surviving, which are two sets of categorical data.

contingency_table = pd.crosstab(df['Pclass'], df['Survived'])
print(contingency_table)
Survived    0    1

grouped = df.groupby(['Pclass', 'Survived']).size().unstack()

fig, ax = plt.subplots(figsize=(10, 6))
grouped.plot(kind='bar', stacked=True, ax=ax)

plt.xlabel('Ticket Class')
plt.ylabel('Count')
plt.xticks(rotation=45)

plt.tight_layout()

res = chi2_contingency(contingency_table)

print("P-value: ", res.pvalue)
P-value:  4.549251711298793e-23

Conclusion 3

Since we got a p-value far below the alpha value of 0.05, then we can REJECT the null hypothesis. Then indeed, we can conclude that the class of a passenger's ticket DOES have an effect on the likelihood that the passenger survives.
